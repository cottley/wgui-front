Navigation

After logging in, you will be presented with a menu on the left hand side of your screen. There are two roles in the system: Admin and User. An Admin level user has access to view and manage users as well as see additional settings.

  • Home provides access for the creation and management of VPN access as an authenticated user.
  • Users is an Admin only function that provides access to manage users, passwords and roles.
  • Settings is an Admin only function that provides access to view the configuration of the front end.
  • Profile provides access to users to self serve themselves to update their information and change their password.
  • About provides information about the software.
  • Logout gives the user the ability to exit the system.
